# BigQuery ML Portfolio Project

## Problem
Predict patients have diabetes or not.

## Tools Used
- Google BigQuery
- BigQuery ML
- SQL
- GitHub

## Model
- Boosted Tree Classifier (BigQuery ML)
- See: docs/model_explanation.md

## Data
- CSV files in /data
- NHANES 1999-2023

## Results
Achieved 86.7% accuracy in churn prediction.

## How to Reproduce
1. Upload CSVs from /data to BigQuery
2. Run SQL in /tables
3. Run model SQL in /models
